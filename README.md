# CosmicUI
> Making old devices feel new again.

CosmicUI is an automated build system that handles the entire firmware process - automatically downloading, unpacking, extracting, modding, and repacking everything back together. 

The main goal of this project is to build a highly portable custom ROM system that can easily use different bases across a bunch of different devices.
